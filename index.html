Solar System

I created a solar system with dynamic physics in OpenGL using C++. OpenGL is a graphics library that lets users create GPU programs called shaders, which are incredibly optimized to run on the GPU and are completely programable, making them very flexible and powerful.

Notable features:

  • Dynamic orbital physics with second order numerical integration
  • WASD, mouse, up, down, and roll movement
  • control over speed of ojbects
  • Blinn-Phong lighting
  • Shadow maps
  • Spotlights
  • Skybox
  • Models
  • Bloom

This project significantly enhanced my system-level understanding of computing, as it involved close interaction with the GPU and necessitated optimizations to achieve a reasonable framerate. A key aspect of my learning was becoming meticulous in managing object pointers, which was essential for robust software architecture. Additionally, this was a creative project for me, boosting my confidence to tackle imaginative projects in the future.

Chatroom

This project involved developing a terminal-based chatroom application in Java, designed to enable communication across various implementations of the app, even if they were developed using different programming languages. This interoperability was facilitated through a standardized protocol, ensuring seamless interaction among users.

Features of the Chatroom:

  • Broadcasting Messages: Users can send messages that are visible to the entire chatroom.
  • User List Retrieval: Allows users to view a list of all participants currently in the chatroom.
  • Private Messaging: Enables users to send private messages to select individuals within the chatroom.

User Interaction Flow:

  • Upon joining the chatroom, users create a username, which triggers a broadcast notification to all participants announcing the new user's arrival.
  • Following this, users can freely engage in conversations with the group or privately with other users.

Project Insights and Achievements

  • Network Programming and Protocol Design: The primary goal was to enhance my understanding of network programming and protocol implementation, crucial for developing applications that communicate over networks.
  • Java Proficiency: A secondary objective was to deepen my expertise in Java, focusing on its application in network communications and concurrent programming.
  • Problem-Solving and Debugging: Throughout the project, I encountered and resolved numerous networking and concurrency issues, which sharpened my problem-solving skills.
  • Project Management: This project also provided valuable experience in managing a self-directed project from conception to execution, furthering my skills in software project management.

Goals Achieved:

By completing this project, I successfully met my objectives of gaining a deeper understanding of system-level concepts through the lens of network programming, enhancing my proficiency in Java, and developing robust problem-solving skills in a real-world application context.
